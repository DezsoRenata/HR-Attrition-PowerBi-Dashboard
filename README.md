The goal of this project was to analyse Hr data and identify patterns that contribute to employee turnover. The dataset consists of employee data, 311 in total of which 104 exited, resulting in an overall attrition of 33%. 
Key insights: 
- Attrition is higher in production and software engineering departments.
- Most turnover is voluntary, driven by better opportunities, dissatisfaction and compensation.
- Attrition mostly peaked between 2015 and 2016, after that it declined, suggesting prior retention efforts had impact.
- Recruitment source has an impact: employee referrals and website hires show lower attrition than job boards.
